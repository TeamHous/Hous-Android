- 오각형 성격 유형 뷰
- 규칙 추가 뷰

- 스플래쉬

- 홈 이벤트 다이얼로그

### 이영주
- 홈 뷰
- 프로필 뷰 (오각형 제외)
- 프로필 수정
- 프로필 설정
